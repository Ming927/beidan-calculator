# 北单奖金计算器

从 [500.com](https://zx.500.com/zqdc/kaijiang.php) 拉取北单（北京单场）真实开奖数据，自动计算全部玩法×全部组合的奖金。

## 快速开始

```bash
pip install -r requirements.txt
python -m src.app
# 浏览器打开 http://localhost:5000
```

## 使用流程

```
① 输入期号（如 26073）→ 获取比赛数据
② 选择场次（默认勾选比分SP最高的15场，支持按玩法SP筛选/随机选）
③ 选择串关级别（如 4关/3关/2关，自动生成 C(n,k) 组合）
④ 点击计算 → 按玩法分组展示所有组合奖金
```

## 支持的玩法

| 玩法 | 选项数 | 最高串关 |
|------|--------|----------|
| 让球胜平负 | 3 (3/1/0) | 15串1 |
| 总进球数 | 8 (0~7+) | 6串1 |
| 上下盘单双数 | 4 | 6串1 |
| 半全场胜平负 | 9 | 6串1 |
| 比分 | 25 | 3串1 |
| 胜负过关 | 2 | 15串1 |

## 奖金规则

- **返奖率**：65%（SP连乘后乘一次）
- **单注封顶**：500万元
- **纳税**：单注奖金超过1万元缴20%个税
- **延期/取消**：该场SP值按1.0计算
- **不能跨玩法串关**：每张票只能选一种玩法

## 项目结构

```
src/
├── models.py            # 数据模型和枚举
├── result_resolver.py   # 赛果判定（7种玩法+让球逻辑）
├── parlay.py            # 串关分解（M串N预设+自由过关）
├── calculator.py        # 核心奖金计算（SP连乘+税收+封顶）
├── zx500_fetcher.py     # 500.com开奖数据解析
├── storage.py           # JSON文件存储
├── app.py               # Flask Web API
└── templates/index.html # 前端页面
tests/                   # 161个测试
```
